# Antigravity Cycling - 極簡藍牙飛輪運動儀表板 (iOS 優化版)

這是一個完全基於網頁瀏覽器的**極簡繁體中文飛輪運動儀表板**。本版本**特別針對蘋果 (iPhone/iOS) 系統、多使用者切換、數據紀錄與智慧運動建議進行了深度優化與功能擴展**。

您無需隨身攜帶電腦，直接用 iPhone 即可連線您的智慧飛輪與 Apple Watch，並能在運動結束後取得即時的智慧踩踏與心率回饋報告。

---

## 🌟 核心特色 (第二階段升級)

*   **iOS 深度優化**：新增自動偵測 iOS 藍牙環境的引導條，支援 iPhone 劉海/動態島與底部 Safe Area 安全區排版。
*   **多使用者設定檔**：支援多個帳戶切換（例如：爸爸、媽媽、小明），各自擁有獨立的姓名與生理資訊（體重、年齡、性別），以大幅提升心率卡路里計算的準確度。
*   **本地運動數據持久化**：所有的騎行紀錄都會自動儲存在您的 iPhone 瀏覽器 (`LocalStorage`) 中，100% 離線可用且絕對隱私。
*   **本週運動量累計**：直覺展示本週運動次數、總時間、總卡路里與總里程。
*   **智慧分析建議引擎**：
    *   **踏頻分析**：自動判定平均踏頻是否太慢，給予齒比與關節保護的力學建議。
    *   **心率強度分析**：自動計算心率是否落在最佳有氧燃脂區，或偏高進入乳酸無氧區，提示調整強度。
    *   **推重比分析**：結合體重與瓦數輸出計算推重比 (W/kg)，給予體能分級。
*   **動態 SVG 單車手**：單車手踩踏與輪子旋轉速度會**百分之百與您的真實踏頻 (RPM) 同步**。
*   **內建模擬器**：開啟「模擬騎行模式」，即可在沙發上模擬踩踏波段，體驗所有功能與動畫。

## 📱 iPhone 連線指引 (iOS 用戶必看)

由於 Apple 原生 **Safari / Chrome 瀏覽器封鎖了 Web Bluetooth 藍牙 API**，為了讓網頁順利連接您的飛輪與 Apple Watch 心率，請務必按照以下步驟操作：

1.  **下載支援藍牙的瀏覽器**：
    *   在您的 iPhone 開啟 App Store。
    *   搜尋並**免費下載「Bluefy」瀏覽器** (或 **WebBLE** 瀏覽器)。
2.  **在 Bluefy 中開啟此網頁**：
    *   在 Bluefy 瀏覽器的網址列輸入您的網頁部署網址（參見下方「雲端發布」或「區域網連線」）。
    *   開啟後，即可點擊「連線飛輪」與「連線心率」進行藍牙連線。

### ❤️ 如何連接 Apple Watch 心率？
1.  在您的 iPhone 與 Apple Watch 上下載第三方廣播軟體（推薦使用免費的 **Echo** 或 **HeartCast**）。
2.  在 Apple Watch 上開啟該 App 並開始運動，這時 App 會將 Watch 的心率轉發成**標準藍牙心率訊號**。
3.  在 Bluefy 瀏覽器開啟的本網頁中，點擊 **「連線心率」**，在彈出的連線視窗中選擇 Echo/HeartCast，即可成功同步心率！

## 🚀 手機直接連線使用方案 (免電腦攜帶)

為了讓您運動時不需要攜帶電腦，我們提供兩種網頁開啟方式：

### 方案 A：一鍵部署至 GitHub Pages (強烈推薦，終身免費且免開電腦)
GitHub Pages 提供免費且安全的 HTTPS 網址，非常適合讓 iPhone 隨時開啟：
1.  將本專案代碼 (`index.html`, `style.css`, `app.js`) 上傳至您的 GitHub 儲存庫 (Repository)。
2.  進入儲存庫的 **Settings > Pages**。
3.  在 **Build and deployment** 下，將 Source 設定為 `Deploy from a branch`，並選取 `main` 分支下的 `/ (root)`，點擊 Save。
4.  幾分鐘後，GitHub 會生成一個安全網址（例如：`https://yourusername.github.io/cycling-app/`）。
5.  在您的 iPhone 上開啟 **Bluefy 瀏覽器**，將此網址加入書籤。以後每次運動，直接點開網址即可配對，數據會自動存在您的 iPhone 中！

### 方案 B：使用局域網 (LAN IP) 連線 (適合電腦與手機在同一個 Wi-Fi 下)
如果您想先在本地測試：
1.  在 Mac 電腦上啟動 Python 伺服器：
    ```bash
    cd "/Users/ashinesun/.gemini/antigravity/scratch/cycling-app"
    python3 -m http.server 8080
    ```
2.  查詢電腦的局域網 IP 位址（在 Mac 「系統設定 > 網路」中查看，例如 `192.168.1.104`）。
3.  將 iPhone 連接至**同一個 Wi-Fi**。
4.  在 iPhone 的 **Bluefy 瀏覽器**中輸入：`http://192.168.1.104:8080` (請將 IP 替換為您的實際電腦 IP) 即可連線。
