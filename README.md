# Antigravity Cycling - 飛輪騎行訓練儀表板 (iOS 優化版)

這是一個完全基於網頁瀏覽器的**極簡繁體中文飛輪訓練儀表板**。本版本以騎行課表、飛輪數據、訓練後健康資料匯入、個人化回饋與訓練計劃為核心。

您可以直接用 iPhone 開網頁連線智慧飛輪。Apple Watch 心率不做即時藍牙連線；改成訓練後用 iPhone 捷徑或手動輸入，將健康摘要匯入本次訓練紀錄。

---

## 🌟 核心特色 (第二階段升級)

*   **iOS 深度優化**：新增自動偵測 iOS 藍牙環境的引導條，支援 iPhone 劉海/動態島與底部 Safe Area 安全區排版。
*   **多使用者設定檔**：支援多個帳戶切換（例如：爸爸、媽媽、小明），各自擁有獨立的姓名與生理資訊（體重、年齡、性別），以提升訓練強度與熱量估算。
*   **本地運動數據持久化**：所有的騎行紀錄都會自動儲存在您的 iPhone 瀏覽器 (`LocalStorage`) 中，100% 離線可用且絕對隱私。
*   **本週運動量累計**：直覺展示本週運動次數、總時間、總卡路里與總里程。
*   **分段騎行課表模式**：自由騎行、恢復騎、有氧耐力、燃脂節奏、穩踩肌耐力、爬坡肌耐力、間歇衝刺、金字塔課表。
*   **踩踏自動開始計時**：連線飛輪後不必手動按開始；偵測到踏頻、功率或速度就自動開始計時。
*   **自動分段阻力**：每個課表都有暖身、主段、高峰或間歇、最後兩分鐘緩和；支援 FTMS 飛輪自動阻力控制。
*   **智慧分析建議引擎**：
    *   **踩踏分析**：結合目前課表、時間、阻力、功率與踏頻，不再只用「轉速太低」單一條件判斷。
    *   **健康匯入分析**：訓練後匯入平均心率、最高心率、主動熱量、運動分鐘、RPE，補完強度判斷。
    *   **推重比分析**：結合體重與瓦數輸出計算推重比 (W/kg)，給予體能分級。
    *   **下次課表建議**：依穩定度、完成度、疲勞感推薦下一次訓練。
*   **動態 SVG 單車手**：單車手踩踏與輪子旋轉速度會**百分之百與您的真實踏頻 (RPM) 同步**。
*   **內建模擬器**：開啟「模擬騎行模式」，即可在沙發上模擬踩踏波段，體驗所有功能與動畫。

## 📱 iPhone 連線指引 (iOS 用戶必看)

由於 Apple 原生 **Safari / Chrome 瀏覽器封鎖了 Web Bluetooth 藍牙 API**，為了讓網頁順利連接您的飛輪，請務必按照以下步驟操作：

1.  **下載支援藍牙的瀏覽器**：
    *   在您的 iPhone 開啟 App Store。
    *   搜尋並**免費下載「Bluefy」瀏覽器** (或 **WebBLE** 瀏覽器)。
2.  **在 Bluefy 中開啟此網頁**：
    *   在 Bluefy 瀏覽器的網址列輸入您的網頁部署網址（參見下方「雲端發布」或「區域網連線」）。
    *   開啟後，即可點擊「連線飛輪」開始訓練。

### 最簡單 iPhone 捷徑匯入設定
先用手動數字版，最快、最穩，不需要做 App。

1.  開啟 iPhone「捷徑」App，新增捷徑，命名 `飛輪健康匯入`。
2.  加入 5 個「要求輸入」動作，類型都選「數字」：
    *   平均心率
    *   最高心率
    *   主動熱量
    *   運動分鐘
    *   RPE 疲勞感 1-10
3.  加入「文字」動作，貼上這段，將每個括號替換成對應的魔術變數：
    ```text
    https://ashinesun2026.github.io/cycling-app/?avgHr=(平均心率)&maxHr=(最高心率)&activeKcal=(主動熱量)&exerciseMin=(運動分鐘)&rpe=(RPE)
    ```
4.  加入「打開 URL」動作，URL 選上一步「文字」。
5.  每次騎完後，先到 Fitness / 健康 App 看摘要，跑這個捷徑填 5 個數字，網頁會自動匯入到最新訓練紀錄。

備用方式：不用捷徑也可以，網頁點「匯入健康」後手動填同樣 5 個數字。

### 平板騎行時的健康匯入
目前資料儲存在當下瀏覽器，不是雲端帳號同步。因此「平板騎行」時，訓練紀錄在平板；手機跑捷徑只會更新手機那份資料。

建議流程：
1.  平板開網頁連線飛輪並完成訓練，按「結束並存檔」。
2.  手機跑捷徑產生帶參數的匯入網址。
3.  用 AirDrop、通用剪貼簿、訊息或備忘錄，把整段網址傳到平板。
4.  平板直接打開該網址，或在平板點「匯入健康」後，把整段網址貼到「捷徑匯入碼」欄位並解析。

若要真正「同帳號登入後自動同步」，後續需要加 Firebase / Supabase 等雲端資料庫；目前靜態網頁版本沒有雲端同步。

## 🚀 手機直接連線使用方案 (免電腦攜帶)

為了讓您運動時不需要攜帶電腦，我們提供兩種網頁開啟方式：

### 方案 A：一鍵部署至 GitHub Pages (強烈推薦，終身免費且免開電腦)
GitHub Pages 提供免費且安全的 HTTPS 網址，非常適合讓 iPhone 隨時開啟：
1.  將本專案代碼 (`index.html`, `style.css`, `app.js`) 上傳至您的 GitHub 儲存庫 (Repository)。
2.  進入儲存庫的 **Settings > Pages**。
3.  在 **Build and deployment** 下，將 Source 設定為 `Deploy from a branch`，並選取 `main` 分支下的 `/ (root)`，點擊 Save。
4.  幾分鐘後，GitHub 會生成一個安全網址（例如：`https://yourusername.github.io/cycling-app/`）。
5.  在您的 iPhone 上開啟 **Bluefy 瀏覽器**，將此網址加入書籤。以後每次運動，直接點開網址即可配對，數據會自動存在您的 iPhone 中！

### 方案 B：使用局域網 (LAN IP) 連線 (適合電腦與手機在同一個 Wi-Fi 下)
如果您想先在本地測試：
1.  在 Mac 電腦上啟動 Python 伺服器：
    ```bash
    cd "/Users/ashinesun/.gemini/antigravity/scratch/cycling-app"
    python3 -m http.server 8080
    ```
2.  查詢電腦的局域網 IP 位址（在 Mac 「系統設定 > 網路」中查看，例如 `192.168.1.104`）。
3.  將 iPhone 連接至**同一個 Wi-Fi**。
4.  在 iPhone 的 **Bluefy 瀏覽器**中輸入：`http://192.168.1.104:8080` (請將 IP 替換為您的實際電腦 IP) 即可連線。
